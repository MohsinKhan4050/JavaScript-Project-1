# ⏰ Digital Clock Web Application

A clean and responsive **Digital Clock Web Application** built using **HTML, CSS, and JavaScript**.
This project demonstrates real-time DOM manipulation, JavaScript timing functions, and responsive layout techniques using Flexbox.

The clock updates every second and dynamically displays **hours, minutes, seconds, and AM/PM format**.

---

## 🚀 Live Features

* ⏱ Real-time digital clock
* 🔁 Automatic time updates every second using `setInterval`
* 🕒 12-hour format with **AM / PM indicator**
* 🎨 Modern UI with background image
* 📱 Responsive layout using **Flexbox**
* ⚡ Lightweight and fast (no frameworks)
